# Line 貼圖自動化助手 (Line Sticker Factory) ![Version](https://img.shields.io/badge/version-1.5.0-06C755?style=flat-square)

這是一個專為 LINE 貼圖創作者設計的自動化工具，旨在簡化從 AI 生圖到最終貼圖打包的流程。透過本工具，您可以輕鬆將 AI 生成的角色網格圖，自動切割、去背並打包成符合 LINE 官方規範的貼圖包。

## ✨ 主要功能

*   **🔀 雙產品類型支援**
    *   **LINE 靜態貼圖**：標準 `370×320` px 貼圖，支援 Main 封面圖 + Tab 標籤圖。
    *   **LINE 表情貼（v1.1.0 新增）**：`180×180` px 正方形表情貼，支援 8/16/24/32/40 張，不含文字、以表情動作傳達情緒。
    *   Header 一鍵切換產品類型，所有 UI、Prompt、切割邏輯自動適配。

*   **🎨 AI 提示詞大師 (Prompt Helper)**
    *   內建多種主題與風格的 Prompt 產生器，支援靜態貼圖與表情貼兩套獨立主題。
    *   **靜態貼圖主題**：日常、打招呼、節日、回應、上班族、老公老婆、迷因搞笑。
    *   **表情貼主題**：日常表情、反應回覆、情緒爆發、可愛萌系、迷因搞笑。
    *   支援 **自訂主題**，可自由填寫文字內容與情緒描述。
    *   **多色文字 Prompt**（靜態貼圖）：自動指定 10 種色系，確保每張貼圖的文字顏色豐富多元、不重複。
    *   **精確尺寸提示**（表情貼）：Prompt 中多次強調精確像素解析度，確保 AI 生成正確尺寸。
    *   **一鍵複製**：Prompt 預覽區右上角內建複製按鈕，方便快速複製到 AI 生圖工具使用。

*   **📐 多網格佈局支援 (Multiple Grid Modes)**
    *   **靜態貼圖**：
        *   4×2（8張）、4×4（16張）、6×4（24張）、4×4 兩組（32張）、6×4+4×4（40張）
    *   **表情貼**：
        *   4×2（8張）、4×4（16張）、6×4（24張）、4×4 兩組（32張）、6×4+4×4（40張）
    *   **多分頁（Double Sheet）模式**：自動處理兩張網格圖的切割與合併，支援混和網格配置。
    *   各主題針對不同網格數量提供對應的文字、情緒與動作建議，確保內容不重複。

*   **✂️ 智能切割 (Smart Slicing)**
    *   自動將上傳的網格圖精準切割為獨立貼圖。
    *   **切割線預覽（v1.1.0 新增）**：上傳圖片後即時顯示紅色虛線網格，讓您在切割前確認對齊狀況。
    *   **圖片尺寸資訊（v1.1.0 新增）**：顯示圖片實際解析度與每格計算尺寸，方便診斷問題。
    *   **整數精度切割**：使用 `Math.round` 獨立計算每格起止點，避免浮點數累積誤差。
    *   即時預覽切割結果。

*   **🪄 專業去背 (Background Removal)**
    *   **綠幕/黑底去背**：針對 AI 生圖常用的綠底或黑底進行優化。
    *   **💎 Gemini 水印自動移除 (v1.3.0 新增)**：採用「反向 Alpha 混合」技術，自動移除 Gemini 生成圖片右下角的 AI 水印，還原純淨背景。
    *   **進階調整**：
        *   **色彩容許度 (Tolerance)**：精準控制去背範圍。
        *   **邊緣柔化 (Smoothness)**：消除鋸齒邊緣。
        *   **溢色去除 (Despill)**：移除邊緣殘留的綠色/黑色光暈。
        *   **裁切縮放**：微調單張貼圖大小，去除不必要的黑邊。

*   **⚡ 自動工作流 (Auto Workflow Mode) (v1.5.0 新增)**
    *   **一鍵壓縮打包**：新增「一鍵產生壓縮檔」功能，自動執行「切割 ➔ 智能去背 ➔ 打包下載」完整流程，無需手動點擊多個按鈕。
    *   **智慧銜接**：自動處理多張圖片的切割與去背過渡，提升創作效率。

*   **📦 一鍵打包 (One-Click Packaging)**
    *   **靜態貼圖**：自動調整至 `370×320` px，支援 Main (240×240) + Tab (96×74)。
    *   **表情貼**：自動調整至 `180×180` px，支援 Tab (96×74)，不需要 Main。
    *   **智慧檔名**：靜態貼圖用兩位數（`01.png`），表情貼用三位數（`001.png`）。
    *   可自訂起始編號，ZIP 檔名自動區分產品類型。

*   **💎 核心美學 (Premium UI/UX)**
    *   **玻璃擬物化設計 (Glassmorphism)**：現代化的透明磨砂質感。
    *   **品牌化視覺**：深度整合 LINE 品牌色彩與 Outfit/Inter 字體。
    *   **雙色主題**：靜態貼圖使用 LINE 綠色系，表情貼使用琥珀色系，視覺區分清晰。
    *   **流暢動畫**：包含滑入、微彈跳與漸變效果，提升創作體驗。

## 🚀 快速開始 (Getting Started)

若您希望在本地端運行此專案，請參考以下步驟：

### 1. 環境需求
確保您的電腦已安裝 [Node.js](https://nodejs.org/) (建議 v16 以上)。

### 2. 安裝依賴
Clone 專案後，進入專案目錄並安裝套件：

```bash
npm install
```

### 3. 啟動開發伺服器
執行以下指令開啟本地開發環境：

```bash
npm run dev
```
啟動後，瀏覽器通常會自動開啟 `http://localhost:5173`。

### 4. 建置生產版本
若要部署至伺服器：

```bash
npm run build
```
建置後的檔案將位於 `dist` 資料夾中。

## 📖 使用教學

### LINE 靜態貼圖

1.  **選擇產品類型**：在 Header 選擇「LINE 靜態貼圖」。
2.  **選擇佈局模式**：選擇網格佈局 — 4×2、4×3、4×4、6×4 或 8×4。
3.  **生成圖片**：展開「AI 提示詞大師」，選擇主題與畫風，複製 Prompt 到 AI 生圖工具。
4.  **上傳與切割**：上傳 AI 生成圖，確認切割線對齊後點擊「開始切割資源包」。
5.  **去背處理**：使用「綠幕去背」或「黑底去背」，視需要調整進階設定。
6.  **打包下載**：選擇 Main + Tab 圖，設定起始編號，下載 ZIP。

### LINE 表情貼

1.  **選擇產品類型**：在 Header 選擇「LINE 表情貼」。
2.  **選擇佈局模式**：選擇數量 — 8/16/24/32/40 張。
3.  **生成圖片**：複製表情貼專用 Prompt（不含文字、強調解析度），到 AI 生圖工具生成。
4.  **確認切割線**：上傳後觀察紅色切割線是否對齊角色邊界。
5.  **去背處理**：執行去背（表情貼使用正方形 180×180 工作畫布）。
6.  **打包下載**：只需選擇 Tab 圖（表情貼不需要 Main），下載 ZIP（檔名為三位數 `001.png`）。
